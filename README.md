# Banking_system
This project illustrate a key banking operations like account creation, balance inquiry, deposit, withdrawal, and transaction history.


# Project Title
Python Banking System Simulation with Transaction Validation

# Project Overview

This project simulates a simple banking system using Python.
It focuses on object-oriented programming and transaction validation.

The system allows users to perform basic operations such as deposits and withdrawals while enforcing rules to prevent invalid transactions.

# Problem Statement

Basic banking operations require proper validation to prevent errors such as overdrawing or invalid deposits.

# This project addresses:

Preventing withdrawals beyond available balance
Ensuring deposits are valid
Maintaining a minimum account balance
Improving transaction logic through structured conditions

# Features

* Account creation with name, number, and balance
* Deposit functionality with validation
* Withdrawal functionality with:
   * Insufficient balance checks
   * Minimum balance enforcement
   * Invalid input handling
   * Clear transaction feedback messages

# Project Structure

* Upgrade 1

* Upgrade 2

* Upgrade 3
